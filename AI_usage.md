# AI_usage

- **Инструмент**: ChatGPT
- **Где использовался**: генерация идей и кода для новых признаков, калибровка параметров модели
- **Новые доработки**: байесовские сглаженные приоры и baseline-прогнозы, латентные факторы через SVD, пользовательские предпочтения по жанрам, авторам, языку, издателю, исправление списка категориальных признаков для CatBoost.
- **Доработка человеком**: предложенные изменения проверены, интегрированы и скорректированы вручную под структуру проекта, оптимизация под Apple Silicon, показатели вовлечённости по wishlist/read, и добавление CatBoost; дополнительный цикл — генерация идей для обогащения временных и жанровых признаков, добавления временного тренда по месяцам и линейной калибровки финального прогноза.
