# AI_usage

- **Инструмент**: ChatGPT (OpenAI, локальное офлайн применение как помощник программирования).
- **Где использовался**: генерация идей и чернового кода для новых признаков (агрегаты по языку/издателю, медианы/стандарты), добавление флагов отключения TF-IDF/BERT, обновление README; последующие шаги — оптимизация под Apple Silicon (детект MPS/NUM_THREADS, батч BERT, многопоточность TF-IDF), усиление модели (временные признаки последней активности/оценки, показатели вовлечённости по wishlist/read, тюнинг LightGBM, опция дообучения на всём train) и добавление CatBoost в ансамбль; дополнительный цикл — генерация идей для обогащения временных и жанровых признаков, исключения текстовых фич из CatBoost, добавления временного тренда по месяцам и линейной калибровки финального прогноза.
- **Данные**: использовались только файлы и схема из репозитория; внешние данные/запросы не применялись.
- **Доработка человеком**: предложенные изменения проверены, интегрированы и скорректированы вручную под структуру проекта.
